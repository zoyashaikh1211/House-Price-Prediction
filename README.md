# 🏠 Pune House Price Prediction

A Machine Learning project that predicts house prices in Pune, India using various regression algorithms.

## 🎯 Project Goal

Build an end-to-end Machine Learning system capable of predicting Pune house prices based on property features such as:

- Location
- Area (Square Feet)
- BHK
- Bathrooms
- Parking
- Property Age
- Floor
- Furnished Status
